# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Easy Ironing, please report it responsibly.

Do NOT open a public GitHub issue.

Instead, contact:

contact@easyironing.com

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact

We will review the report and respond as soon as possible.

## Scope

This repository covers:
- The website
- The Telegram bot backend
- Related infrastructure code

## Notes

Do not attempt to exploit vulnerabilities beyond proof of concept.
Responsible disclosure is appreciated.
